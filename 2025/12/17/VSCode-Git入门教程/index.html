VS Code + Git 入门教程

本教程将带你从零开始，学会使用 VS Code 和 Git 将代码上传到 GitHub。

1. 下载安装

1.1 安装 VS Code

  1. 访问官网：https://code.visualstudio.com/
  2. 点击 Download for Windows
  3. 运行安装程序，一路点击 下一步
  4. 勾选以下选项：
    • ✅ 将”通过 Code 打开”操作添加到文件上下文菜单
    • ✅ 将”通过 Code 打开”操作添加到目录上下文菜单
    • ✅ 添加到 PATH

1.2 安装 Git

  1. 访问官网：https://git-scm.com/downloads
  2. 点击 Windows 下载
  3. 运行安装程序，大部分选项保持默认即可
  4. 安装完成后，打开终端验证：

如果显示版本号（如 git version 2.43.0），说明安装成功。

7. 创建 GitHub 仓库并推送

7.1 在 GitHub 创建新仓库

  1. 登录 GitHub
  2. 点击右上角 + → New repository
  3. 填写仓库名称（如 my-first-project）
  4. 选择 Public（公开）
  5. 不要 勾选 “Add a README file”
  6. 点击 Create repository

命令 说明
git init 初始化仓库
git status 查看状态
git add . 添加所有文件

git push 推送到远程
git pull 拉取远程更新
git clone 地址 克隆仓库
git log 查看提交历史
